# PRD - 船长清单（ShipMate）v0.1

## 1. 产品定位
- 类型：AI 生产力工具（ToDo + Focus + Reflection）
- 目标用户：有目标管理需求的个人创作者、独立开发者、职场人
- 价值主张：每天只做三件最重要的事，用 AI 帮你复盘与迭代

## 2. 核心场景
1) 早晨：创建今日 3 个最重要任务
2) 白天：用番茄钟推进任务，记录中断原因
3) 晚间：一键生成复盘（完成度、阻碍、改进建议）

## 3. MVP 功能清单
### 3.1 今日三件事
- 新增/编辑/完成任务
- 任务字段：标题、预计时长、优先级、备注
- 限制：最多 3 个活跃 MIT

### 3.2 专注计时
- 25/5 默认，可配置 15/25/50
- 会话结束后可选择关联任务
- 统计：今日专注时长、会话数

### 3.3 AI 复盘
- 输入：今日任务完成情况 + 专注统计 + 用户补充文字
- 输出：
  - 今日总结（100-150字）
  - 明日建议（3条可执行动作）
  - 风险提醒（1-2条）

### 3.4 数据层
- 本地离线可用
- 导出 JSON（MVP）

## 4. 非功能要求
- 启动时间 < 2s
- 离线可用
- 崩溃率目标 < 1%
- 首版仅中文

## 5. 商业化
- Free：每日 1 次 AI 复盘
- Pro（订阅）：不限次数、复盘模板、周报导出

## 6. 上架合规要点
- 明确隐私政策 URL
- 明确数据收集项（如接入分析）
- 若接入账号体系，需提供账号删除路径

## 7. 里程碑
- M1（D1-D3）：UI + 本地任务/计时
- M2（D4-D7）：AI 复盘闭环
- M3（D8-D10）：设置页、导出、埋点
- M4（D11-D14）：测试、截图、提审材料
